# Builder Pattern [Patrón creacional] [GangOfFour]
## ¿Para qué se usa? 
Inicializar una clase compleja con muchas propiedades

## ¿Cómo funciona?
Haciendo la creación paso a paso, muy útil cuando se requieren muchos pasos o cuando se requiere un órden

## ¿Cómo funciona técnicamente?
Creando una clase separada llamada Builder, para crear el objeto en vez de usar el constructor. Cuando el Builder termina su construcción devuelve el objeto creado al cliente.
